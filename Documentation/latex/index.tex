Welcome to Water Supply Analysis tool! This tool is designed to analyze the water supply network and provide insights into maximizing water distribution efficiency, identifying critical areas, and evaluating network resilience.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Objectives}{Objectives}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item Implementing a water supply management analysis tool using greedy algorithmic approaches, particularly in the context of maximum-\/flow problems.
\item Creating a user-\/friendly interface for showcasing functionalities and conducting short presentations.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Description}{Description}}\label{index_autotoc_md3}
The water supply network system provided in this project consists of various elements such as water reservoirs, pumping stations, and delivery sites. The task is to develop functionalities to address two main sets of questions\+:


\begin{DoxyItemize}
\item \mbox{[}x\mbox{]} Basic Service Metrics\+: Determine maximum water flow to each city, check if the existing network configuration meets water demand, and balance load across the network.
\item \mbox{[}x\mbox{]} Reliability and Sensitivity to Failures\+: Evaluate network resilience by simulating failures of water reservoirs, pumping stations, and pipelines, and analyze the impact on water supply to cities.
\end{DoxyItemize}\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Dataset}{Dataset}}\label{index_autotoc_md4}
The dataset stores in CSV files and includes information about water reservoirs, pumping stations, delivery sites, and pipelines of Portugal and Madeira.\hypertarget{index_autotoc_md5}{}\doxysubsection{\texorpdfstring{Data Handling}{Data Handling}}\label{index_autotoc_md5}
